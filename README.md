## Overview

SpiderFoot is a custom HackHub terminal command for discovering mod content from websites, Twotter profiles, and registered network targets.
This tool was originally created for my own mod, but I decided to release it for other authors that are looking for a tool they can adjust to their own specific modding needs. 

Using SpiderFoot's backend, you are able to enrich networks and website with additional information that can then be surfaced by the player with a SpiderFoot search. You can also control exactly which information you want to surface and how it should match.

Check out the full documentation complete with examples and a quick start guide here: https://darkvalnar.github.io/hackhub-spiderfoot/
